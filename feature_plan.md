# Feature Plan

## Phase 0: Foundations (Current)

- Monorepo scaffold
- shared types package
- OpenAI integration baseline
- health + extraction starter endpoints

## Phase 1: Wedge Product

- Upload pipeline (image/pdf)
- extraction schema templates (receipt, invoice)
- validation engine (totals, mandatory fields)
- human review queue UI
- export to CSV/webhook

## Phase 2: Team Workflow

- role-based access
- reviewer assignment + audit logs
- dataset/version snapshot for reviewed examples
- confidence and drift dashboard

## Phase 3: Flowstate Parity Layer

- no-code workflow builder
- active learning loop
- edge deployment adapters
- multi-tenant org controls
